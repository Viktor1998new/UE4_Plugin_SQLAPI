#Installation

copy DLL files from ThirdParty\SQLAPI\lib
  -sqlapi.dll
  -sqlapid.dll
  -sqlapiu.dll
  -sqlapiud.dll
also need a SQL client DLL, MySQL has libmysql.dll
to the NameProject\Binaries\Win64

(in the build of the game, also copy all the DLL files from the list, including the SQL client)
